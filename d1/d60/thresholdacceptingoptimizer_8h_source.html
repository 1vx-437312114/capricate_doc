23 /**
24  * @class ThresholdAcceptingOptimizer
25  * @brief Optimizes the wirelength of a path by using the threshold acceptance algorithm.
26  *
27  * Threshold Accepting is a local search method similiar to Hill Climbing and Simulated
28  * Annealing. A local search method starts with a random solution and then
29  * explores its neighbourhood in the solution space by moving (usually randomly) from its
30  * current position to a neighborhood, accepting a new solution if and only if it improves
31  * the objective function. It can therefore get stuck in local minima / maxima. Threshold accepting
32  * overcomes this problem by also allowing uphill-moves / downhill-move, that it also accepts
33  * new solutions which lead to higher objective function values. But deteriorations are only accepted
34  * if they are not worse than a particular threshold. Over time, the threshold decreases to zero,
35  * thus Threshold Accepting turns into a classical hill climbing algorithm.
36  *
37  */
