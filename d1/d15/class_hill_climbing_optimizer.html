Detailed Description

Optimizes the total wirelength of all connections by using the hill climbing algorithm.

The HillClimbingOptimizer is an iterative algorithm that starts with an arbitrary solution to a problem, then attempts to find a better solution by making an incremental change to the solution (called a neighborhood). If the change produces a better solution, another incremental change is made to the new solution, and so on until no further improvements can be found or a number of iterations has been exceeded.

Random-restart hill climbing is a meta-algorithm built on top of the hill climbing algorithm. It iteratively does hill-climbing, each time with a new random initial condition. If a new run of hill climbing produces a better result than the until then best stored state, it replaces the best stored state.

Hill climbing will not necessarily find the global minimum, but may instead converge on a local minimum.
