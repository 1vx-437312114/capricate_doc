32 //! \brief Declaration of a generic QuickSort function.
33 //!
34 //! The jqsort() function sorts an array with numels elements.
35 //! The v argument points to the start of the array of elements casted to void *.
36 //! The contents of the array are sorted in ascending order according to a
37 //! comparison function pointed to by comp, which is called with two
38 //! arguments that point to the objects being compared.
39 //! The comparison function must return an integer less than, equal to, or
40 //! greater than zero if the first argument is considered to be respectively
41 //! less than, equal to, or greater than the second. If two members
42 //! compare as equal, their order in the sorted array is undefined.
43 //! See the manpage of the standard library qsort() function for further information.
