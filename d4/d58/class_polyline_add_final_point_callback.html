Detailed Description

Callback for the polyline touchpoint generation tool.

This callback is executed when the user presses the left mouse button. It is derived from vtkCommand and is an implementation of the observer/command design pattern.

To select an area on the surface of the mesh, defined by a polyline contour, the user has has to define the vertices of the contour line by pressing the left mouse button at every vertex position. After defining the polyline on the mesh, the surface area of the touchpoint is created by reconstructing the mesh surface inside the marked area complemented by the contour line of the selection. Following the reconstruction, the newly created touchpoint geometry is added to the scene for rendering and to the touchpoint registry for further processing.
