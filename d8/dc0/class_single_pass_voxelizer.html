Detailed Description

Single pass rasterization based voxelizer.

The SinglePassVoxelizer voxelizes a mesh via rasterization. For small voxel grid resolutions it is by far the fastest of all the voxelizers and gives very good results for artifact free input meshes. A disadvantage of the single pass voxelizer is that it is prone to degenerations of the input mesh.
