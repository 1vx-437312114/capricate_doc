163 /**
164  * @class LassoInteractorStyle
165  * @brief Interactor style for lasso tool.
166  *
167  * This class implements the majority of motion control routines and defines an event driven
168  * interface callback which is executed when the user presses the left mouse button.
169  *
170  * To draw a lasso selected area on the surface of the mesh, the user has
171  * to keep the left mouse button pressed down while drawing a contour line.
172  * After releasing the mouse button, the surface area of the touchpoint is
173  * created by reconstructing the mesh surface consisting of the triangles of
174  * the mesh inside the marked area. Following the reconstruction, the newly
175  * created touchpoint geometry is added to the scene for rendering and to the touchpoint
176  * registry for further processing.
177  */
