Detailed Description

3D-Printing construction component which is not of type wiring, touchpoint or connector.

This class represents additional 3D-printing construction components which are not of type wiring, touchpoints, connectors or mesh and which can also contain further subparts. It is a compound which can consist of several parts of geometry, touchpoints and magnets.
