Detailed Description

The RandomOptimizer generates random connections in the hope to find a good one by pure luck.

The RandomOptimizer is a highly inefficient optimization algorithm based on the generate and test paradigm. The function successively generates random permutations of solutions. If a generated solution is better than the current best solution, it is stored. It is not useful for optimizing, but may be used to contrast it with more efficient algorithms. It also may even not find a feasible solution at all.
