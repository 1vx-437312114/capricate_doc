25 /**
26  * @class HillClimbingOptimizer
27  * @brief Optimizes the wirelength of a path by using the hill climbing algorithm.
28  *
29  * The HillClimbingOptimizer is an iterative algorithm that starts with an arbitrary
30  * solution to a problem, then attempts to find a better solution by making an incremental
31  * change to the solution (called a neighborhood). If the change produces a better solution,
32  * another incremental change is made to the new solution, and so on until no further improvements
33  * can be found or a number of iterations has been exceeded.
34  *
35  * Random-restart hill climbing is a meta-algorithm built on top of the hill climbing algorithm.
36  * It iteratively does hill-climbing, each time with a new random initial condition. If a new run of
37  * hill climbing produces a better result than the until then best stored state, it replaces the
38  * best stored state.
39  *
40  * Hill climbing will not necessarily find the global minimum, but may instead converge on a local minimum.
41  *
42  */
