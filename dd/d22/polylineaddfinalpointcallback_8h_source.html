103 /**
104  * @class PolylineAddFinalPointCallback
105  * @brief Callback for the polyline touchpoint generation tool.
106  *
107  * This callback is executed when the user presses the left mouse button. It is
108  * derived from vtkCommand and is an implementation of the observer/command design pattern.
109  *
110  * To select an area on the surface of the mesh, defined by a polyline contour,
111  * the user has has to define the vertices of the contour line by pressing the
112  * left mouse button at every vertex position. After defining the polyline on the mesh,
113  * the surface area of the touchpoint is created by reconstructing the mesh surface
114  * inside the marked area complemented by the contour line of the selection.
115  * Following the reconstruction, the newly created touchpoint geometry is added to the
116  * scene for rendering and to the touchpoint registry for further processing.
117  */
118 class PolylineAddFinalPointCallback : public vtkCommand
119 {
120  private:
121  // fields
