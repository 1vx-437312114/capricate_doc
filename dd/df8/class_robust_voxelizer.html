Detailed Description

Robust voxelizer based on raycasting and majority voting.

This robust voxelizer uses raycasting to classify voxels belonging to the inside or outside of a mesh. It is based on a generalization from the Jordan curve theorem in 3D. The Jordan curve theorem asserts that every Jordan curve divides the plane into an "interior" region bounded by the curve and an "exterior" region containing all of the nearby and far away exterior points, so that every continuous path connecting a point of one region to a point of the other intersects with that loop somewhere.
