Detailed Description

Connection between a source touchpoint and a destination connector.

This class represents a connection between a source (touchpoint) and a destination (connector). It consists of the geometry represented through a vtkPolydata object, the graphical representation (vtkActor), a vector of indices of the cells which are traversed in the path from source to destination, a pointer to the source touchpoint and destination connector and a boolean flag to signal if the 26-voxel neighborhood of each path voxel is marked (locked) too.
