Detailed Description

Optimizes total wirelength of all connections by using the threshold acceptance algorithm.

Threshold Accepting is a local search method similiar to Hill Climbing and Simulated Annealing. A local search method starts with a random solution and then explores its neighbourhood in the solution space by moving (usually randomly) from its current position to a neighborhood, accepting a new solution if and only if it improves the objective function. It can therefore get stuck in local minima / maxima. Threshold accepting overcomes this problem by also allowing uphill-moves / downhill-move, that it also accepts new solutions which lead to higher objective function values. But deteriorations are only accepted if they are not worse than a particular threshold. Over time, the threshold decreases to zero, thus Threshold Accepting turns into a classical hill climbing algorithm.
