Detailed Description

Does not optimize at all (null) but instead connects the first touchpoint with the first connector and so on.

The NullOptimizer does not optimize at all but instead connects the first touchpoint with the first connector, the second touchpoint with the second connector and so on. It is useful for forcing specific connections by hand.
