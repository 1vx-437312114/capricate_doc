Detailed Description

Tool for drawing arbitrary shaped touchpoints on a mesh.

To draw a freehand area on the surface of the mesh, the user has to keep the left mouse button pressed down while drawing a contour line. After releasing the mouse button, the surface area of the touchpoint is created by reconstructing the mesh surface inside the marked area complemented by the contour line of the selection. Following the reconstruction, the newly created touchpoint geometry is added to the scene for rendering and to the touchpoint registry for further processing.
