Detailed Description

The ConnectionInfo struct contains the indices of the source and destination touchpoint.

The ConnectionInfo struct contains the indices of the source and destination touchpoint / connector, a vector of 3D-indices of the path of the connection inside the 3D-Grid, a boolean flag markNeighborHood which determines if the 26-neighborhood of each voxel of the path is also marked (locked) for other connections.
