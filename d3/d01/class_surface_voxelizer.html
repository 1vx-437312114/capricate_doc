Detailed Description

Mesh surface only voxelizer.

The surface voxelizer only voxelizes the outer shell of a mesh. It does this via rasterization of triangles into the 3D voxel grid. The algorithm has a running time of O(n) (where n is the number of triangles) and is fast.
