Detailed Description

Multipass rasterization based robust voxelizer.

The MultiPassVoxelizer does three single pass voxelization passes based on rasterization. In each pass the rasterization is carried out along a different axis (x, y, z). The results of the three passes is combined via majority voting to a final image. If rendering artifacts occur during the rasterization step they occur most of the time only in one axis direction (often because of degenerated triangles or normals) and not in the direction of the other axes. With the multi pass / majority voting approach these artifacts vanish most of the time making the result more stable. A disadvantage of the algorithm is that it takes a bit more than three times the amount of time compared to a single pass algorithm.
