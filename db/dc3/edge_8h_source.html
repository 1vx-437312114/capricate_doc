40 //! Edge of a Basic_TMesh.
41 
42 //! This class represents an edge of a triangulation. An Edge is the main
43 //! part of the Basic_TMesh data structure. Each edge has an orientation
44 //! (i.e. from v1 to v2) which forces the order in which incident triangles
45 //! (t1 and t2) are stored in the class. When looking the edge so that it
46 //! points "upwards", if the normal of t1 points towards the observer then
47 //! t1 must be on the left of the edge. The field mask is useful for
48 //! assigning up to 256 different states to the edge.
