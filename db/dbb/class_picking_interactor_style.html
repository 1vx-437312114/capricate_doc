Detailed Description

Interactor style for placing sphere shaped (or other fixed geometry) touchpoints tool.

This class implements the majority of motion control routines and defines an event driven interface callback which is executed when the user presses the left mouse button.

It places a sphere shaped touchpoint on the mesh. The newly created touchpoint is added to the scene for rendering and to the touchpoint registry for further processing.
