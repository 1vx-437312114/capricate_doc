Detailed Description

Filter for boolean operations on polygon meshes using the vtk library with the vtkBooleanOperationPolyDataFilter.

The VTKMeshBooleanOperationFilter is a filter for applying boolean operations on polygon meshes using the vtk library with the vtkBooleanOperationPolyDataFilter.
